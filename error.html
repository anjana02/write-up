Walkthrough:

  5. Navigate to the Postman public workspace URL by adding the root domain to the workspace details disclosed: https://www.postman.com/equipment-inventory/workspace/

  6. There is an existing API collection within the Postman URL discovered. Analyze the details of the API endpoints.
  7. Navigate to the API endpoint to fetch the account details and observe the HTTP request headers configured.
  8. The final flag will be in the HTTP headers section of the API endpoint.
