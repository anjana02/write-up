Description: In the heart of Adversary City, a critical web application server has fallen into the hands of a notorious APT group. This cunning adversary leveraged a sophisticated web application attack to breach the server's defenses. Now, the investigation team is left with a single clue: a log file, brimming with cryptic traces of the attack. Your mission is to decipher the logs, uncover the attack vector, discover the removed code and track down the elusive APT group to capture the coveted flag. Will you rise to the challenge and outsmart the adversaries? The fate of Adversary City rests in your hands!

Walkthrough:


  1. Navigate through the Log file provided as part of the challenge.


  2. Identify the NPM package Webpack-AltEsc being updated. There is an indication in the logs that the NPM package was not found in the initial update.
  3. Note the indication in the logs that the NPM package was not found during the initial update.
  4. The attack type used here is called “Dependency Confusion.”

  5. The APT group strategically pushed a malicious NPM package instead of the legitimate one, targeting the public repository.


  6. The application during a routine update, fetched the malicious NPM package hosted by the attacker and this resulted in the compromise.


  7. A snapshot the NPM package is available on Way back machine in case of the NPM package is removed by the scanners
